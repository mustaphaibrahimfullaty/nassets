# Project Memory

This project is frontend-first.

Important:

1. Everything uses mock data.
2. No real backend yet.
3. All components must be reusable.
4. Architecture must support future APIs.
5. Investor demo quality is critical.
6. Dark mode is mandatory.
7. Mobile experience is first class.
8. All pages should feel premium.
9. Avoid hardcoding values inside components.
10. Build every feature as if production APIs already exist.

Always use:

- TypeScript
- Interfaces
- Repositories
- Service Layer
- Design Tokens
